# Top 10 Post-MVP Features

1. Location-aware guide discovery with country, region, and city landing pages.
2. Admin repeatable field editor for documents, steps, fees, FAQ, and sources.
3. Editorial review workflow with assignments, due dates, and reviewer notes.
4. Expiration reminders when guide review dates are older than a configured threshold.
5. User reminders for appointments, renewals, and submission deadlines.
6. Guide comparison for requirements that vary by country, state, city, or institution.
7. Source change monitoring that flags official pages when content changes.
8. Document vault metadata, without storing sensitive file contents by default.
9. Team or family workspaces for shared school, business, or immigration checklists.
10. Public feedback loop so users can report unclear, outdated, or missing requirements.
